# Magna Carta: Voices Against Absolute Power

Static website for an NHC National History Competition project on the theme
"Voices."

## Open Locally Forever

This website does not need a server to exist. Open `index.html` directly in a
normal browser such as Chrome, Safari, or Edge:

```text
/Users/komronabdurakhmanov/Documents/Codex/2026-06-12/im-doing-nhc-national-history-competition/index.html
```

The Codex in-app browser may block direct `file://` pages, but normal browsers
can open the file.

## Make It Permanently Public

Use GitHub Pages for a permanent public URL:

1. Create a new public GitHub repository.
2. Upload everything in this folder, including `index.html`, `styles.css`,
   `script.js`, `.nojekyll`, and the `assets` folder.
3. In GitHub, open the repository settings.
4. Go to Pages.
5. Set the source to deploy from the main branch and the root folder.
6. Save. GitHub will give you a public website URL.

The site is already written as plain static HTML/CSS/JavaScript, so it is ready
for GitHub Pages, Netlify, Cloudflare Pages, or any other static website host.
